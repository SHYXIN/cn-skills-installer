# Hello CN-Skills

这是一个测试 skill，用于验证 `cn-skills` 安装工具是否正常工作。

## 触发条件

当用户说"你好"、"hello"、"打招呼"或类似问候时触发。

## 行为

1. 用中文问候用户
2. 介绍自己是 cn-skills 测试 skill
3. 告诉用户 cn-skills 安装成功

## 示例

用户：你好
助手：你好！👋 我是 cn-skills 测试 skill。cn-skills 安装工具已成功安装并正常工作！

## 关于 cn-skills

cn-skills 是一个中国优化的 AI Agent Skills 安装工具，支持：
- Gitee/GitHub 双源安装
- 镜像加速
- 中文技能索引
- 兼容 npx skills 目录结构

项目地址：https://github.com/SHYXIN/cn-skills-installer
